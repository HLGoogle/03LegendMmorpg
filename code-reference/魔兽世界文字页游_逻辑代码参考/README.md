# 魔兽世界：巫妖王之怒 - 文字RPG

一款基于《魔兽世界：巫妖王之怒》主题的文字RPG游戏，使用HTML、CSS和JavaScript开发，所有数据保存在浏览器本地存储中。

## 🎮 游戏特色

### 🎭 角色系统
- **5个独特职业**：战士、法师、猎人、圣骑士、牧师
- **80级等级上限**，每个职业都有专属属性和技能树
- **最多5人组队**，支持多角色协同作战
- **角色死亡机制**，使用金币复活

### ⚔️ 战斗系统
- **回合制战斗**，采用经典的MMO战斗逻辑
- **坦克-治疗-输出（T-H-D）**系统
- **仇恨值系统**，坦克角色吸引怪物攻击
- **自动战斗和手动战斗**选项
- **技能冷却机制**，策略性战斗

### 🛡️ 装备系统
- **5种品质**：普通（白）、优秀（绿）、精良（蓝）、史诗（紫）、传说（橙）
- **9个装备槽位**：武器、头盔、胸甲、护腿、手套、靴子、戒指x2、项链
- **一键装备功能**，自动为角色装备最优物品
- **装备强化**，基于品质有不同强化次数上限
- **附魔系统**，消耗灵魂之尘提升装备属性

### 🗺️ 副本系统
- **10个副本**，从低级到高级：
  - 哀嚎洞穴（10级）
  - 死亡矿井（15级）
  - 血色修道院（25级）
  - 剃刀高地（35级）
  - 黑石深渊（45级）
  - 通灵学院（55级）
  - 黑石塔（60级）
  - 安其拉废墟（60级）
  - 熔火之心（60级，团队副本）
  - 冰冠堡垒（80级，团队副本）
- **普通怪物和BOSS**，BOSS有独特掉落
- **挑战次数限制**，可升级上限
- **传说装备掉落**，熔火之心和冰冠堡垒

### ⚒️ 专业系统
- **钓鱼**：获得鱼材料
- **采矿**：获得矿石材料
- **附魔**：分解装备获得灵魂之尘
- **珠宝**：炸矿将矿石转化为宝石

### 💪 强化系统
- **攻击强化**：每级增加固定攻击力
- **防御强化**：每级增加防御和生命值
- **速度升级**：优化战斗节奏（0-10级）
- **最高9999级**攻击和防御强化

### 🏆 成就系统
- **15项成就**，涵盖击杀、通关、收集等
- **永久属性加成**，完成成就提升全队属性
- **成就进度跟踪**

## 🎯 使用方法

1. 打开 `index.html` 文件即可开始游戏
2. 点击"添加角色"创建你的第一个角色
3. 选择职业并命名
4. 选择副本开始战斗
5. 收集装备、升级角色、完成成就

## 💾 存档功能

- **自动保存**：游戏数据保存在浏览器本地存储
- **手动保存**：点击"保存"按钮
- **导出存档**：将存档导出为文本文件
- **导入存档**：从文本文件导入存档

## 🎨 界面说明

### 顶部导航
- 金币和团队人数显示
- 保存、导出、导入、重置、帮助按钮

### 左侧面板
- 团队成员信息
- 添加角色按钮
- 一键装备按钮

### 中间面板（标签页）
- **冒险**：副本列表
- **战斗**：战斗日志和战斗控制
- **装备**：装备背包和出售功能
- **技能**：角色技能展示
- **专业**：生活技能和专业技能
- **强化**：攻击、防御、速度强化
- **成就**：成就列表和进度

### 右侧面板
- 团队总属性
- 材料数量
- 成就加成

## 🔧 技术实现

- **前端**：HTML5、CSS3、JavaScript（ES6+）
- **存储**：LocalStorage API
- **响应式设计**：支持桌面和移动设备
- **暗色主题**：模仿魔兽世界风格

## 📝 注意事项

1. 游戏完全在浏览器本地运行，无需服务器
2. 清除浏览器缓存会丢失游戏进度
3. 建议定期导出存档备份
4. 传说装备只在特定副本BOSS掉落

## 🎮 游戏技巧

1. **职业搭配**：建议至少1个坦克和1个治疗
2. **副本选择**：根据队伍平均等级选择合适副本
3. **装备策略**：优先升级攻击和强化，提高刷本效率
4. **成就优先**：完成成就获得永久加成
5. **资源管理**：合理分配金币用于复活、强化、附魔
