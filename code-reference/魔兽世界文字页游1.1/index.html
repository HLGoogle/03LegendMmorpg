📖 游戏帮助

🎭 职业

10大职业可选：战士(坦克/输出)、圣骑士(坦克/治疗/输出)、猎人(输出)、潜行者(输出)、牧师(治疗/输出)、萨满祭司(治疗/输出)、法师(输出)、术士(输出)。每个职业拥有3种天赋专精，适应不同战斗角色。

🏹 种族

10大种族可选：人类、矮人、侏儒、暗夜精灵、德莱尼、兽人、巨魔、牛头人、亡灵、血精灵。每个种族提供独特的属性加成和专业技能增益。

⚔️ 战斗系统

回合制战斗，坦克吸引仇恨，治疗恢复生命，输出造成伤害。挑战副本首领可获得史诗装备。自动战斗功能解放双手，高效刷装备。

🛡️ 装备系统

装备品质：普通(白)、优秀(绿)、精良(蓝)、史诗(紫)、传说(橙)。高品质装备提供更强属性。可通过战斗掉落、任务奖励、副本通关获取。

🗺️ 野外冒险

9大区域逐级挑战：北郡、艾尔文森林、西部荒野、赤脊山、暮色森林、湿地、地狱火半岛、虚空风暴、冰冠冰川。每个区域有对应等级怪物和任务。

📜 任务系统

完成任务获得经验、金币和装备奖励。任务类型多样：击杀怪物、收集物品、护送NPC等。完成主线任务解锁新区域和副本。

⚒️ 专业技能

钓鱼：获取鱼类食材。采矿：采集矿石。附魔：分解装备获得灵魂之尘。珠宝加工：将矿石转化为宝石。专业技能可提升角色属性。

💪 强化系统

攻击强化：提升全队攻击力。防御强化：提升全队防御和生命值。速度强化：提升战斗速度，最高10级。消耗金币进行升级。

🏆 成就系统

完成各类成就获得永久属性加成，如攻击加成、防御加成、生命加成。成就涵盖击杀、装备、副本、专业等多个方面。

🏰 副本挑战

挑战地下城副本，击败强大首领，获取稀有装备和材料。副本难度逐级提升，需要合理团队配置和策略。

🎒 材料收集

收集鱼、矿石、灵魂之尘、宝石等材料，用于专业技能和装备提升。材料可通过专业活动获得。

⚙️ 游戏设置

可调节游戏倍率：装备掉率、金币掉率、经验获取、属性成长。上帝模式提供高级功能，方便自定义游戏体验。
